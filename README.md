# ATech Open-Use Proprietary License (AOUPL)

This repository is the **official source and reference implementation** of the ATech Open-Use Proprietary License (AOUPL).

## Purpose

The AOUPL defines a custom software distribution model created by ATech that allows:

- unrestricted use of software in binary form
- unrestricted redistribution of unmodified binaries
- commercial usage and resale of distribution/support
- while maintaining **closed-source ownership and no modification rights**

This repository exists to:

- define the canonical license text
- track updates and revisions to the license
- provide a stable reference for projects using AOUPL

## License Status

The license itself is maintained in this repository under:

- [`LICENSE`](./LICENSE.md)

That file is considered the **authoritative version** of the AOUPL.

## Versioning

The AOUPL is versioned. Each update will:

- increment the version number (e.g., v1.0 → v1.1)
- be documented in commit history and/or release tags
- preserve backwards compatibility where possible

## Usage of This Repository

This repository does **not** contain software for end-user consumption.

Instead, it serves as:

- a legal reference source
- a version-controlled license definition
- a public transparency record of changes

## Key Principles

The AOUPL is based on the following principles:

- Software should be freely usable and distributable
- Ownership of implementation remains private
- Redistribution should not require modification rights
- Commercial ecosystems around software distribution are permitted
